# Assets

The tomato artwork is **original**, generated by
[`scripts/make_icon.py`](../../scripts/make_icon.py).

| File(s)                     | Purpose                                            |
|-----------------------------|----------------------------------------------------|
| `tomatick.icns`             | App bundle icon (Finder, Cmd-Tab)                  |
| `mb_red.png` / `mb_white.png` / `mb_black.png` | Idle menu-bar icon for each selectable theme |
| `mb_<theme>_0.png … _N.png` | Alarm frames (shake + pulse) played while an event is alarming |
| `about.png`                 | Large tomato shown on the Settings → About tab     |
| `extlink.png`               | External-link glyph on the About link buttons      |
| `bmc.png`                   | Buy Me a Coffee button — Buy Me a Coffee's official button image, used to link to the maintainer's page |
| `octocat.png` *(optional)*  | GitHub's logo for the "GitHub Repo" button — **not bundled**; drop GitHub's official mark here to show it, otherwise the button falls back to text + external-link |

The menu-bar theme (red / white / black) is chosen in **Settings → General →
Menu bar icon**. Red is a glossy tomato with a shine; white and black are solid
silhouettes for light/dark menu bars.

## Regenerate

```bash
python scripts/make_icon.py
```

This rewrites `tomatick.icns`, all `mb_*.png` frames, `about.png` and
`extlink.png`. Edit the colors, shape, or alarm frames (`ALARM_FRAMES`) in
`make_icon.py`. (`bmc.png` is downloaded from Buy Me a Coffee, not generated.)
