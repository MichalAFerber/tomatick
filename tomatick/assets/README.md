# Assets

All icons here are **original artwork**, generated by
[`scripts/make_icon.py`](../../scripts/make_icon.py) — no third-party assets.

| File(s)                     | Purpose                                            |
|-----------------------------|----------------------------------------------------|
| `tomatick.icns`             | App bundle icon (Finder, Cmd-Tab, About)           |
| `mb_red.png` / `mb_white.png` / `mb_black.png` | Idle menu-bar icon for each selectable theme |
| `mb_<theme>_0.png … _N.png` | Shake-animation frames played while an event is alarming |

The menu-bar theme (red / white / black) is chosen in **Settings → General →
Menu bar icon**. Red is a bright tomato with a shine; white and black are solid
silhouettes for light/dark menu bars.

## Regenerate

```bash
python scripts/make_icon.py
```

This rewrites `tomatick.icns` and all `mb_*.png` frames. Edit the colors, shape,
or shake angles (`SHAKE_ANGLES`) in `make_icon.py`.
